## チューインガムとは

チューインガムは、噛むことを目的とした菓子の一種。主成分は水に溶けないガムベース（合成ゴムまたは天然樹脂）で、糖分、香料、軟化剤を含む。

**基本構成:**
- ガムベース: 60%（水不溶性）
- 甘味料と香料: 40%（水溶性）

## 歴史

### 古代の起源（3世紀〜）

**マヤ文明（紀元3世紀頃）**
- サポジラ（チューインガムノキ）の樹液を煮て固めたものを噛む習慣
- 中央アメリカ（メキシコ南部、グァテマラ、ホンジュラス）のアステカ族とマヤ族が発祥
- この樹液を「チクル（chicle）」と呼ぶ

### 近代ガムの誕生

**1860年代: サンタ・アナとの出会い**
- メキシコ大統領アントニオ・ロペス・デ・サンタ・アナがアメリカに亡命
- サンタ・アナはチクルをゴム代替品として売り込もうとした
- 失敗に終わったが、チクルがアメリカに持ち込まれる契機となる

**1869年: トーマス・アダムスによる発明**
- Thomas Adams（トーマス・アダムス）がサンタ・アナの秘書として働く
- チクルをゴム製品（タイヤ、長靴、おもちゃ）に使おうと試みるが失敗
- ある日、薬局で少女がパラフィンワックスガムを1ペニーで買うのを見る
- サンタ・アナがチクルを噛んでいたことを思い出し、ガムとして商品化を思いつく

**1871年: 商業的成功**
- "Adams New York No. 1" Gumを発売（無糖、1ペニー）
- 同年、世界初のフレーバー付きガム "Black Jack"（リコリス味）を発売
- 1888年: 初の自動販売機をニューヨーク地下鉄に設置
- 1899年: American Chicle Company の初代会長に就任

### 日本への導入

**1916年（大正5年）**
- 日本に初めて輸入されるが、普及せず

**1928年（昭和3年）**
- 日本国内での生産開始

**戦後（1945年〜）**
- 進駐軍の米兵がガムを噛む姿が格好良く映り、若者の間で流行
- ロッテなどの企業が市場を牽引

## 成分と構造

### ガムベース（Gum Base）

チューインガムの心臓部。水に溶けず、弾力性を提供する。

#### 主要成分

**1. エラストマー（Elastomers）- 弾力性の提供**
- ポリイソブチレン（Polyisobutylene）
- イソブチレン-イソプレン共重合体（Butyl rubber）
- スチレン-ブタジエンゴム（Styrene-butadiene rubber）
- 天然チクル（Chicle - サポジラ樹脂）

**2. 樹脂（Resins）- 強度と凝集性**
- 酢酸ビニル樹脂（Polyvinyl acetate）- ガムベースの約60%
- エステルガム（Ester gum - マツヤニ由来）
- テルペン樹脂（Terpene resins）

**3. 軟化剤（Waxes/Plasticizers）**
- パラフィンワックス（Paraffin wax）
- 微結晶ワックス（Microcrystalline wax）

**4. 充填材（Fillers）**
- 炭酸カルシウム（Calcium carbonate）
- タルク（Talc）

**5. 乳化剤（Emulsifiers）**
- レシチン（Lecithin）

### 水溶性成分

**甘味料:**
- 砂糖（糖入りガム）
- キシリトール（Xylitol）
- ソルビトール（Sorbitol）
- アスパルテーム（Aspartame）

**香料:**
- ペパーミントオイル
- スペアミントオイル
- フルーツフレーバー

**その他:**
- 保存料
- 着色料
- 酸味料

### 成分の役割

```
ガムの構造:

[ガムベース層]（水不溶性）
  ↑ 弾力性、形状維持

[水溶性層]（甘味料・香料）
  ↑ 味、香り
  → 唾液に溶けて口から消える

噛み続けると...
水溶性成分が抜ける → 味がなくなる
ガムベースだけ残る → ゴム状に硬化
```

### 化学的特性

**ガムベースの重要な性質:**

1. **ガラス転移温度: 約29℃**
   - 29℃以下: ガラス状（硬い）
   - 29℃以上: ゴム状（柔らかい）
   - 体温37℃で柔軟性を発揮

2. **非極性ポリマー**
   - 水に不溶
   - 油脂に可溶

3. **規制上の分類**
   - アメリカFDA: 46種類の化学物質を「ガムベース」として承認
   - 具体的な配合は企業秘密

## 製造プロセス

### 工場での製造（完全自動化）

チューインガムは人の手に触れることなく、清潔で空調管理された施設で製造される。

#### ステップ1: ガムベースの調製

```
原材料の投入
  ↓
加熱・溶解（高温で溶かす）
  ↓
精製（不純物除去）
  ↓
ベース材料の準備完了
```

#### ステップ2: 混合（Mixing）

```
大型ミキサー（最大1トン）
  ↓
1. ガムベース投入
2. 甘味料追加
3. 香料投入（タイミングが重要）
4. その他添加物
  ↓
30-60分間混合
  ↓
大きな「ガムの塊」完成
```

#### ステップ3: 圧延（Rolling）

```
ガムの塊
  ↓
ローラー1（粗い）
  ↓
ローラー2（中間）
  ↓
ローラー3（細かい）
  ↓
薄く広いリボン状に
```

各ローラーペアは前のペアより近く設定され、徐々に厚みを減らす。

#### ステップ4: 切断/成型（Cutting/Scoring）

```
連続リボン状のガム
  ↓
切断機
  ↓
製品タイプ別に切断:
- スティックガム（長方形）
- タブレットガム（短い塊）
- ロール状ガム（長いストリップ）
- コーティング用芯（小さい長方形）
```

#### ステップ5: 冷却（Cooling）

```
温度管理室
  ↓
24-48時間冷却
  ↓
ガムの固化と安定化
```

#### ステップ6: コーティング（Candy-coated gumの場合）

```
ガム芯を回転ドラムに投入
  ↓
糖シロップを噴霧
  ↓
乾燥
  ↓
次の層を噴霧
  ↓
（繰り返し）
  ↓
硬い糖衣完成
```

例: Wrigley's Extra, Dentyne Ice

#### ステップ7: 包装（Packaging）

```
高速包装機
  ↓
気密包装（鮮度保持）
  ↓
箱詰め
  ↓
出荷
```

## ガムが溶ける理由

### 基本原理: "Like Dissolves Like"（似たものは似たものを溶かす）

ガムベースは**非極性ポリマー**。
- 水（極性）には溶けない
- 油脂（非極性）には溶ける

### 溶けるメカニズム

#### 1. 油脂による溶解

**化学的説明:**

```
ガムベースの分子構造:
CH₂-CH₂-CH₂-CH₂-...（ポリイソブチレン）
    ↑ 非極性の長い鎖

油脂（トリグリセリド）:
R-COO-CH₂
    ↑ 非極性部分

非極性同士 → 親和性が高い
    ↓
ガムベースが油脂に溶ける
```

**油脂がガムベースに作用すると:**
- 分子間結合が切れる
- 網目構造が崩壊
- ガムが液体化

**溶けやすい食品:**
- チョコレート（ココアバター含有）
- バター
- サラダ油
- ナッツ類
- 油っぽい食事

#### 2. 温度による溶解

**ガラス転移温度: 29℃**

```
温度が上がると:
29℃以下 → ガラス状（硬い）
29℃-40℃ → ゴム状（柔軟）
40℃以上 → 軟化（溶け始める）

体温: 37℃ → すでに軟化している
熱い飲み物: 60-80℃ → さらに溶ける
```

**溶けやすい飲み物:**
- 熱いコーヒー
- 紅茶
- ホットチョコレート
- スープ

#### 3. チョコレートとガムの反応

**なぜチョコレートでガムが急速に溶けるのか？**

```
チョコレートの成分:
- ココアバター（油脂）30-40%
- 乳脂肪（ミルクチョコの場合）
    ↓
強力な溶解作用

メカニズム:
1. ココアバターがガムベースに浸透
2. ポリイソブチレンの分子鎖が切断
3. 網目構造が崩壊
4. ガムが液状に
5. 唾液と混ざって流れる
```

実験:
```
口の中でガムを噛む
  ↓
チョコレートを一口
  ↓
数秒後...
  ↓
ガムが消失！
```

#### 4. 唾液による微量溶解

**通常の唾液でもガムが徐々に溶ける理由:**

唾液の成分:
- 水: 99.5%
- 脂質: 0.01-0.1%（微量だが存在）
- タンパク質
- 電解質

```
長時間（1時間以上）噛み続けると:
微量の脂質が蓄積
  ↓
ガムベースが少しずつ溶ける
  ↓
ガムが小さくなる、または柔らかくなる
```

**個人差の理由:**
- 唾液中の脂質量は人によって異なる
- 食事内容（油っぽいものを食べた後）
- 体温（口腔内温度が高い人ほど溶けやすい）

#### 5. 溶けないとき

**水だけでは溶けない:**

```
水分子: H-O-H（極性）
    ↑
    ↓ 反発
ガムベース（非極性）
    ↓
溶解しない
```

通常の唾液（99.5%が水）では、ガムベースは溶けずに残る。

## 健康への影響

### 利点（Benefits）

#### 1. 口腔衛生の向上

**唾液分泌の促進:**
- 通常の唾液分泌量: 0.3-0.5 mL/分
- ガム噛み時: 3-6 mL/分（10-12倍）

**唾液の効果:**
```
唾液分泌増加
  ↓
口腔内のpH上昇（酸性→中性）
  ↓
虫歯菌（Streptococcus mutans）の活動抑制
  ↓
歯のエナメル質再石灰化
  ↓
虫歯予防
```

**キシリトールガムの特別な効果:**
- 虫歯菌の増殖を直接抑制
- プラーク（歯垢）の形成を防ぐ
- 虫歯リスクを最大30-40%低減（研究結果）

**アメリカ歯科医師会（ADA）の推奨:**
- 食後20分間、シュガーフリーガムを噛む
- ADA承認マーク付きガムを選ぶ

#### 2. 集中力と記憶力の向上

**脳への血流増加:**
```
咀嚼運動
  ↓
顎の筋肉収縮
  ↓
脳への血流 +25-40%
  ↓
酸素と栄養素の供給増加
  ↓
脳の活性化
```

**研究結果:**
- 短期記憶テストのスコアが35%向上（カーディフ大学研究）
- 注意力・判断力・認識力の改善
- 反応時間の短縮

#### 3. ストレス軽減

**メカニズム:**
- 咀嚼がリズミカルな運動 → リラックス効果
- コルチゾール（ストレスホルモン）の低減
- 不安・抑うつの軽減

**研究データ:**
- 長期的なガム噛みで、うつ・不安・ストレスが軽減
- 学業成績の向上も報告

#### 4. その他の利点

- 食欲抑制（ダイエット効果）
- 口臭予防
- 禁煙支援（ニコチンガム）
- 耳管機能の改善（飛行機での耳抜き）

### 欠点（Drawbacks）

#### 1. 顎関節症（TMD: Temporomandibular Disorder）

**過度な咀嚼による問題:**
```
長時間・過度の噛み
  ↓
顎の筋肉疲労
  ↓
顎関節への過剰な力
  ↓
TMDの発症
```

**症状:**
- 顎の痛み
- 口を開けるときのカチッという音
- 頭痛
- 顔面痛

**リスク群:**
- 1日数時間以上噛む人
- 既に顎の問題がある人

#### 2. 虫歯のリスク（糖入りガム）

```
糖入りガム
  ↓
口腔内に糖が持続的に供給
  ↓
虫歯菌が酸を産生
  ↓
歯のエナメル質を溶かす
  ↓
虫歯
```

**推奨:** シュガーフリーガムのみ使用

#### 3. 消化器系の問題

**糖アルコール（キシリトール、ソルビトール）の副作用:**
- 大量摂取（1日10-15粒以上）→ 下痢
- 腹部膨満感
- ガス

**理由:** 糖アルコールは小腸で完全に吸収されず、大腸で発酵する

#### 4. 歯の損傷

過度な力での噛み:
- 詰め物の脱落
- 歯の摩耗
- 歯の破折（稀）

#### 5. その他の懸念

- 頭痛（過度な咀嚼による）
- 誤飲のリスク（特に小児）
- 中毒性（心理的依存）

### 推奨事項

**安全なガムの噛み方:**
1. シュガーフリーガムを選ぶ
2. 1回の咀嚼時間: 15-20分程度
3. 1日の総時間: 1-2時間まで
4. 顎に痛みがあれば中止
5. 食後20分が最適なタイミング

**避けるべき状況:**
- 顎関節症の既往がある
- 詰め物・被せ物が多い
- 妊娠中（一部の添加物）
- 幼児（誤飲リスク）

## 環境問題

### ガム = プラスチック汚染

#### 深刻な現状

**年間生産量:**
- 世界: 3740億個のガム
- 総重量: 10万トン

**廃棄の現状:**
- 80-90%が不適切に廃棄される
- 路上に捨てられたガムは世界で最も多いポイ捨てゴミの一つ

**環境への影響:**
```
捨てられたガム
  ↓
生分解されない（プラスチックだから）
  ↓
数十年かけて光分解
  ↓
マイクロプラスチック化
  ↓
雨で海へ流出
  ↓
海洋汚染
```

### なぜガムは環境問題なのか

#### 1. 合成プラスチック製

現代のガムの主成分:
- ポリイソブチレン（車のタイヤと同じ）
- 酢酸ビニル樹脂（接着剤と同じ）
- ポリエチレン（レジ袋と同じ）

**これらは石油由来の合成ポリマー**

#### 2. 生分解性がない

```
プラスチックガムの分解時間:
- 完全分解: 数百年〜永久
- 表面硬化: 数時間
- マイクロプラスチック化: 数十年
```

天然ゴムと違い、微生物では分解できない。

#### 3. 除去コストが高い

**イギリスの例:**
- 年間清掃費用: 6000万ポンド（約110億円）
- 路上のガム除去: 1個あたり1.50ポンド（約270円）

**除去方法:**
- 高温スチーム
- 特殊な溶剤
- 手作業でのこすり取り

**ロンドンのオックスフォード・ストリート:**
- 1平方メートルあたり300個以上のガムが付着

#### 4. 海洋への流出

```
路上のガム
  ↓
雨で下水道へ
  ↓
処理されずに河川へ
  ↓
海へ流出
  ↓
マイクロプラスチックとして蓄積
  ↓
海洋生物が誤飲
```

### 世界各国の対策

#### シンガポール

**1992年: ガムの販売・輸入禁止**
- 理由: 地下鉄のドアにガムが貼られ、運行に支障
- 罰金: 最高10万シンガポールドル（約1100万円）+ 懲役
- 2004年: 医療用ガムのみ解禁（歯科医の処方箋が必要）

#### アメリカ

**一部の都市で罰金:**
- カリフォルニア州の一部: 路上にガムを吐くと250ドルの罰金
- ニューヨーク: ポイ捨て禁止条例

#### ヨーロッパ

**環境意識の高まり:**
- ドイツ: ガム専用ゴミ箱の設置
- オランダ: 生分解性ガムの推奨
- フランス: 環境に優しいガムの市場拡大

## 生分解性ガムの台頭

### 新世代ガムの特徴

#### 天然チクル使用

**チクル（Chicle）とは:**
- サポジラ（Manilkara zapota）の樹液
- メキシコ・中央アメリカ原産
- 持続可能な収穫が可能

**特性:**
- 100%天然
- 生分解性: 3-4週間で分解
- 化学処理不要

#### 主なブランド

**1. Chicza（チクサ）**
- メキシコ産
- オーガニック認証
- フェアトレード

**2. True Gum（トゥルーガム）**
- デンマーク発
- 100%プラントベース
- プラスチックフリー包装

**3. Simply Gum（シンプリーガム）**
- アメリカ
- 天然成分のみ
- 6種類の原材料のみ使用

**4. Glee Gum（グリーガム）**
- アメリカ
- チクル使用
- 非遺伝子組み換え

**5. Chewsy（チューシー）**
- イギリス
- 完全ビーガン
- 生分解性包装

### 従来品との比較

| 項目 | 合成ガム | 生分解性ガム |
|------|---------|------------|
| 主成分 | 酢酸ビニル樹脂、ポリイソブチレン | チクル、天然樹脂 |
| 分解時間 | 数百年 | 3-4週間 |
| 環境負荷 | 高い | 低い |
| 価格 | 安い（100円/パック） | 高い（300-600円/パック） |
| 食感 | 長持ち | やや短い |
| 味の持続 | 長い | 短い |

### 課題

**価格:**
- 天然チクルの収穫コストが高い
- 大量生産が困難

**食感:**
- 合成ポリマーほどの弾力性がない
- 味の持続時間が短い

**普及:**
- 認知度が低い
- 流通が限定的

### トリビア

**1. ガムを飲み込んでも大丈夫？**
- 答え: 基本的には無害
- ガムベースは消化されずに排出される
- 7年間体内に残るという俗説は誤り
- 大量に飲み込むと腸閉塞のリスク（稀）

**2. 宇宙でガムを噛める？**
- 答え: 可能だが推奨されない
- 無重力下で唾液が飲み込みにくい
- 飛び散るリスクがある

**3. ガムの歴史的使用**
- 第二次世界大戦: 兵士にガムが配給された（ストレス軽減目的）
- ベトナム戦争: ガムが正式な軍需品に

**4. ガムと犯罪捜査**
- ガムからDNAが採取可能
- 犯罪現場のガムから犯人特定の事例多数

**5. ガム禁止の場所**

**6. ガムの特許**

## まとめ

チューインガムは3世紀のマヤ文明から始まり、1869年のトーマス・アダムスの発明で近代化され、現在は年間10万トン以上が生産される巨大産業となった。

**主な特徴:**
- ガムベース（合成ゴム）が主成分
- 油脂と高温で溶ける（水には溶けない）
- シュガーフリーガムは口腔衛生に有益
- 過度な噛みは顎関節症のリスク

**環境問題:**
- 合成ガムはプラスチック汚染の一因
- 年間10万トンが路上や海洋に廃棄
- 生分解性ガムが新たな解決策として台頭

**未来:**
- 天然チクルを使った環境に優しいガムへのシフト
- 健康意識と環境意識の高まり
- テクノロジーによる改良（味、食感、分解性）

チューインガムは、単なる嗜好品を超えて、化学、健康、環境、文化が交差する興味深い製品である。
